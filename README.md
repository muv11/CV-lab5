# CV-lab5
### Задание: Развернуть нейронную сеть на архитектуре MobileNet, детектирующую объекты на видео. Рекомендуемые технологии - keras + MobileNet (https://keras.io/api/applications/mobilenet/) предобученный на imagenet. В качестве входных данных можно использовать видео-файл и входной поток с вебкамеры.
